# 后台

  1. REST API: Koa/Hapijs + mongodb + redis
    hapijs 是一个开源的、基于 Node.js 的应用框架,由 walmart(沃尔玛) 技术团队开发的 RestAPI 框架，经过沃尔玛网站、电子商城实用，框架稳定、可靠。支持 ES6 async/await,可以通过插件方式让开发灵活，模块清晰。
    MongoDB 是一个高性能，开源，无模式的文档型数据库，Mongodb 使用 C++开发，保证用户高扩展性，可以灵活支持用户修改数据属性。
    我们采用全栈 JavaScript,ES6 的 async/await 方式编写代码，保证代码的模块代和可读性，保证用户的定制化开发效率及维护性。

  2. CMS: Vue + ElementUI
    前后端分离的 CMS
    多角色权限管理

  3. 阿里云 CentOs7 64x
